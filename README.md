# MediaFlow 🚀

MediaFlow is a modern web application built with Next.js that allows users to upload, preview, download, and transform media files. It supports video playback directly in the browser and enables image conversion into popular social media formats such as Instagram thumbnails, Facebook posts, and more.

## Features

• Upload videos and images
• Preview videos directly in the browser
• Download uploaded media files
• Convert images to different social media formats
• Automatic format naming and resizing
• Fast and responsive UI
• Secure authentication
• Cloud media storage

## Tech Stack

* Next.js
* React
* Prisma
* Cloudinary
* Clerk Authentication
* Tailwind CSS

## Use Cases

This tool is useful for:

* Content creators
* Social media managers
* Marketing teams
* Developers building media tools

Users can quickly generate correctly sized images for platforms like Instagram, Facebook, and YouTube without manually editing them.
